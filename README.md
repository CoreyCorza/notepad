# Notepad
A premium, multi-tabbed notepad application built with Electron.

## Features
- **Multi-Tab Support**: Open multiple files simultaneously in a sleek tabbed interface.
- **Custom Themes**: Fully customizable UI colors via the Preferences panel.
- **Dark Mode Confirmation**: Custom, beautiful dark-themed confirmation modals—no more white system dialogs.
- **Session Restoration**: Remembers everything—open files, untitled notes, and unsaved changes—exactly where you left off.
- **OS Integration**: "Open with Notepad" right-click context menu for `.txt` and `.md` files.
- **Drag & Drop**: Drop files anywhere on the window to open them instantly.
- **Word Wrap**: Persistent toggle for long lines in the View menu.
- **Single Instance**: Smart instance locking—opening new files uses your existing window.
- **Custom Icon**: Bespoke notebook icon integration.
- **Modern UI**: Powered by Lucide SVG icons and high-end aesthetics.

## Installation
1. Download the latest `Notepad Setup.exe` from the releases.
2. Run the installer to add Notepad to your system and context menu.
